# 🐍 Nokia Snake Game

A classic Nokia snake game recreated as a modern web application! This is the famous game from old Nokia mobile phones, now playable in your browser.

## Features

- 🎮 **Classic Gameplay** - The original snake mechanics you know and love
- ⌨️ **Keyboard Controls** - Arrow keys or WASD to move
- 📱 **Mobile Support** - Swipe controls for touch devices
- 📊 **Score Tracking** - Compete for the highest score
- 🏞️ **Responsive Design** - Plays perfectly on desktop and mobile
- 💚 **Nokia Aesthetic** - Classic Nokia green game board

## How to Play

1. Click "Start Game" to begin
2. Guide the snake to eat the red food
3. Each food eaten increases your score by 10
4. Avoid hitting the walls or yourself
5. Try to get the highest score possible!

## Controls

### Desktop
- **Arrow Keys** or **WASD** - Move the snake
- **Mouse Click** - Start/Pause/Resume game

### Mobile
- **Swipe** - Change direction
- **Tap** - Start/Pause/Resume game

## Game Rules

- The snake starts with 3 segments
- Move the snake towards the red food to eat it
- Each food eaten makes the snake longer
- Game ends if you hit a wall or yourself
- Score increases by 10 points for each food eaten

## Installation

1. Clone this repository or download the files
2. Open index.html in your web browser
3. Start playing!

### Online Play
Enable GitHub Pages in repository settings to play online.
